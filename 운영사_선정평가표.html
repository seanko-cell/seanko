📋 활용 안내

  1. 섹션 1에서 각 항목 Pass(O) / Fail(X)를 선택하세요. 하나라도 X 시 해당 운영사는 평가 제외됩니다.
  2. 섹션 2 노란색 셀에 지역별·품목별 배송 운임(원)을 입력하면 가상 바구니 점수가 자동 산출됩니다.
  3. 섹션 3 노란색 셀에 값을 입력하면 점수가 자동 계산됩니다. ⑤는 S / A / B로 입력.
  4. 섹션 4에서 종합 점수와 판정 결과를 확인하세요. 70점↑ 선정 검토 / 60점 미만 탈락

📋 활용 안내

  1. 각 운영사별로 월간 KPI 실측값을 입력하면 점수가 자동 계산됩니다.
  2. 소비자 불만 건수는 건수 그대로 입력하면 KPI 점수에서 자동 차감됩니다 (건당 -1점).
  3. 노출 순위 산정 섹션에서 수수료·배송·KPI 점수를 입력하면 가중 합산 순위가 산출됩니다.
